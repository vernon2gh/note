# xv6是一个简单的类Unix教学操作系统

## 简介

在2006年夏天，xv6作为一个教学操作系统进行开发，现在我们移植xv6到Risc-V架构，作为一门新本科课堂 6.S081

## xv6源码与书

最新的xv6源码与书，可以通过如下得到：

```bash
$ git clone git://github.com/mit-pdos/xv6-riscv.git
$ git clone git://github.com/mit-pdos/xv6-riscv-book.git
```
