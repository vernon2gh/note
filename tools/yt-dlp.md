# 简介

Command-line program to download videos from YouTube.com and other video sites.

# 安装

```bash
$ apt install yt-dlp ffmpeg
```

# 使用

```bash
$ yt-dlp [OPTIONS] URL
```

可选参数：

* `--list-formats`：列出可用的视频格式，如：分辨率
* `--cookies-from-browser chrome`：指定加载 chrome 浏览器的 cookies
